description: 'Тодорхой бараа материалын шаардлагыг ID-аар авах'
---

## Endpoint

<RequestExample>
```bash cURL
curl -X GET 'https://your-odoo-instance.com/api/product-expense/123' \
  -H 'Cookie: session_id=your_session_id'
```
</RequestExample>

## Path Parameters

<ParamField path="expense_id" type="integer" required>
Бараа материалын шаардлагын ID
</ParamField>
