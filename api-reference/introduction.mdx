## Тавтай морил

MW Portal Backend API нь Odoo 18 системтэй холбогдох REST API-уудыг өгдөг. Энэ API нь portal хэрэглэгчдэд зориулсан бөгөөд ажилтны профайл, ирц, чөлөөний хүсэлт, бараа материалын шаардах, худалдан авалтын хүсэлт зэрэг үйлчилгээнүүдийг агуулдаг.

## Authentication

Бүх API endpoint-үүд нь Odoo session-based authentication ашигладаг. Хэрэглэгч нь Odoo системд нэвтэрсэн байх ёстой бөгөөд session cookie эсвэл session ID-г ашиглана.

<Warning>
Бүх endpoint-үүд `auth='user'` тохиргоотой бөгөөд нэвтэрсэн хэрэглэгч шаардлагатай.
</Warning>

## Response Format

Бүх API хариу нь дараах форматтай:

<CodeGroup>
```json Success Response
{
  "success": true,
  "data": {...},
  "message": "Амжилттай"
}
```

```json Error Response
{
  "success": false,
  "message": "Алдааны мэдээлэл"
}
```
</CodeGroup>
