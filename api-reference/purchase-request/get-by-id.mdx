## Endpoint

<RequestExample>
```bash cURL
curl -X GET 'https://your-odoo-instance.com/api/purchase-request/123' \
  -H 'Cookie: session_id=your_session_id'
```
</RequestExample>

## Path Parameters

<ParamField path="request_id" type="integer" required>
Худалдан авалтын хүсэлтийн ID
</ParamField>

| 200 | Амжилттай |
