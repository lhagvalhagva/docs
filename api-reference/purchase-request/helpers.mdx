## Branches

<RequestExample>
```bash cURL
curl -X GET 'https://your-odoo-instance.com/api/purchase-request/branches' \
  -H 'Cookie: session_id=your_session_id'
```
</RequestExample>

Салбаруудын жагсаалт авах.
