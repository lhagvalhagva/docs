## Endpoint

<RequestExample>
```bash cURL
curl -X POST 'https://your-odoo-instance.com/web/session/authenticate' \
  -H 'Content-Type: application/json' \
  -d '{
    "jsonrpc": "2.0",
    "params": {
      "db": "your_database_name",
      "login": "your_username",
      "password": "your_password"
    }
  }'
```
</RequestExample>

## Request Body

<ParamField body="jsonrpc" type="string" required>
JSON-RPC хувилбар. Үргэлж `"2.0"` байна.
</ParamField>

<ParamField body="params" type="object" required>
Параметрүүдийг агуулсан объект.

<Expandable title="Params Fields">
  <ParamField body="db" type="string" required>
  Database-ийн нэр
  </ParamField>

  <ParamField body="login" type="string" required>
  Хэрэглэгчийн нэвтрэх нэр (username/email)
  </ParamField>

  <ParamField body="password" type="string" required>
  Хэрэглэгчийн нууц үг
  </ParamField>
</Expandable>
</ParamField>
