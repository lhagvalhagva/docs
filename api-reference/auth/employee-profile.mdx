## Endpoint

<RequestExample>
```bash cURL
curl -X GET 'https://your-odoo-instance.com/api/auth/employee-profile' \
  -H 'Cookie: session_id=your_session_id'
```
</RequestExample>

## Authentication

Энэ endpoint нь Odoo session-based authentication ашигладаг. Хэрэглэгч нь Odoo системд нэвтэрсэн байх ёстой.

## Status Codes

| Status Code | Description |
|-------------|-------------|
| 200 | Амжилттай |
| 400 | Session ID шаардлагатай |
| 404 | Ажилтан олдсонгүй |
| 500 | Серверийн алдаа |
